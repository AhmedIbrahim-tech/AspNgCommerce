#. Change Color

1. Change material colors

In src/styles.scss file, line no. [15, 16, 17] contain [primary, accent, warn]. Check material color palette, and change it'

      
        // ---
        $lieb-primary: mat.define-palette(mat.$green-palette, 700, A400, A700);
        $lieb-accent: mat.define-palette(mat.$deep-orange-palette, 500, 400, A700);
        $lieb-warn: mat.define-palette(mat.$yellow-palette, 700, 500, A700);
        // ---
2. Change Tailwindcss colors

In your project-directory/tailwind.config.ts file, line no. 22 contain colors object, change color primary, accent, warn color with colors code. you can add more color by tailwindcss colors, Check Tailwindcss color customization

      
        {{
          "theme: {
            // ---
            colors: {
              primary: '#58B57D',
              accent: '#EE5E36',
              warn: '#FDE300',
              black: colors.black,
              white: colors.white,
              gray: colors.coolGray,
              red: colors.red,
              blue: colors.blue
            }
          }"
        }}
3. Change theme colors

In src/assets/scss/_variables.scss file, line no. 1, 2, 3 contain primary, accent, warn colors

      
        $primary: #00BCD4;
        $accent: #FF5722;
        $warn: #FBC02D;
